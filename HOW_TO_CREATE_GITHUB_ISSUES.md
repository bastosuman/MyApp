## ğŸš€ Quick Steps to Create Issues

### Option 1: Manual Copy-Paste (Recommended)

1. Go to your GitHub repository: `https://github.com/bastosuman/MyApp`
2. Click on **"Issues"** tab
3. Click the green **"New issue"** button
4. For each issue:
   - Click "New issue"
   - Open the corresponding `.github/issues/issue-X-*.md` file in your project
   - Copy the entire content
   - Paste into the GitHub issue form
   - **Title**: Use the title from the first line (without the #)
   - **Labels**: Add these labels:
     - Issue 1: `enhancement`, `database`, `backend`
     - Issue 2: `enhancement`, `api`, `backend`
     - Issue 3: `enhancement`, `api`, `backend`
     - Issue 4: `enhancement`, `api`, `backend`
     - Issue 5: `enhancement`, `frontend`, `api`, `integration`
   - Click **"Submit new issue"**
5. Repeat for all 5 issues

### Option 2: Use GitHub Web Editor

1. Go to your repository on GitHub
2. Navigate to `.github/issues/` folder
3. For each file, click on it and view the raw content
4. Copy the content to create a new issue
